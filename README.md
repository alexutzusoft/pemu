# Pemu

**A lightweight, cross-platform processor emulator CLI in C supporting 52 distinct CPU architectures.**

---

## Overview

Pemu is a modular, high-performance processor emulator written in C. Designed for developers, computer architecture enthusiasts, and retro gaming fans, it provides a generic interface to load binaries, execute instructions step-by-step, and emulate historical microprocessors and microcontrollers.

## Features

- **52 Supported CPU Cores** — Emulates architectures ranging from historical 4-bit chips (Intel 4004) to modern 64-bit cores (RISC-V RV64I, ARM7TDMI).
- **Interactive CLI & Stepping** — Real-time execution speed control, cycle delay tweaking, and step-by-step disassembly debugging.
- **Embedded Console & Graphics Support** — Native display rendering for NES (MOS 6502) and microcontroller hardware peripherals (ATmega328P UART & Timers).
